# AI Qualification Dashboard

A small React / JavaScript demo project created for learning and portfolio purposes.

The idea is to build a simple dashboard for checking workers, qualifications and certificate status.

## Features

- list of workers
- qualification status
- certificate expiration status
- filtering by valid, expired and missing qualifications
- simple dashboard UI
- local JSON data

## Why I built this

I created this project to show my frontend learning path, product thinking and AI-assisted development workflow.

I am interested in building real product features, learning fast and using AI as a thinking tool, not only as a code generator.
